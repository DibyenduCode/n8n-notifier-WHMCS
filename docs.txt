📘 n8nnotifier – WHMCS Addon Module Documentation
1. Introduction

The n8nnotifier module is a custom WHMCS Addon Module designed to display notifications (such as login/register alerts, announcements, or custom messages) directly in the WHMCS Client Area.
It also provides an Admin Configuration Panel to manage notifications, making it easier for administrators to control what clients see.

3. Installation Guide

Follow these steps to install the addon:

Upload Files
Copy the folder n8nnotifier into your WHMCS addons directory:

/path-to-whmcs/modules/addons/n8nnotifier/


Activate the Module

Log in to WHMCS Admin Area

Navigate to:
Setup → Addon Modules

Find n8nnotifier in the list

Click Activate

Configure Permissions

Once activated, click Configure

Assign which Admin Roles can access the module.

Database Setup

On activation, the module creates necessary database tables (if coded in n8nnotifier.php).

On deactivation, these tables may be dropped. (This ensures cleanup.)

Verify Installation

You should now see a menu item in the Addons → n8nnotifier section.

9. Troubleshooting
❌ Error: Too few arguments to function select_query()

WHMCS 8+ requires 3+ arguments for select_query.

Ensure your database queries follow:

select_query('table', 'fields', ['conditions']);

❌ Notifications not showing

Ensure hooks are properly registered in n8nnotifier.php.

Check tplfileupdatedcode contains valid HTML/Smarty code.

Clear WHMCS template cache.

❌ Module not activating

Verify folder path: /modules/addons/n8nnotifier/

File permissions: 644 for PHP, 755 for folders.
